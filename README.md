# API - Usuarios

## Descripción

Proyecto desarrollado en JavaScript que consume una API REST utilizando XMLHttpRequest para obtener y administrar información de usuarios.

Endpoint utilizado:
https://jsonplaceholder.typicode.com/users

## Funcionalidades

- Listar nombres de usuarios
- Buscar información básica de usuario
- Mostrar dirección completa
- Mostrar información avanzada
- Listar compañías y catchPhrase
- Ordenar usuarios alfabéticamente

## Ejecución

1. Abrir archivo index.html
2. Abrir consola del navegador (F12)
3. Presionar los botones disponibles
